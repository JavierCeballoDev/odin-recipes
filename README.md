# odin-recipes

Este es el primer proyecto que abordo en lo que respecta al currículum dispuesto por la comunidad The Odin Project.

Se trata de un sitio web básico de recetas que consta de una página de índice principal con enlaces a varias de mis recetas favoritas.

Mediante este proyecto pongo en práctica varios temas revisados hasta ahora, como control de versiones con Git/GitHub y conceptos básicos de HTML.
